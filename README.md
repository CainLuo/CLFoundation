## CLFoundation简介

`CLFoundation`是对系统`Foundation API`进行扩展的库, `API`的用法全部都在`CLFoundationExample`里, 如果在使用库遇到的一些疑惑欢迎随时`Issues`给我.

支持`CocoaPod`管理

> pod 'CLFoundation'
